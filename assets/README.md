## Projekt felépítés
- A projekt felépítése rendezett, minden a maga helyén van.
```
├───ikt
│   │   index.html
|   |   bankito.html
|   |   pecsito.html
|   |   tisza.html
|   |   vadasato.html
│   │
│   └───assets

```
- Az összes kép az adott tó-ról megadott mappában vannak.
